PANDUAN APLIKASI SISTEM INFORMASI EKSTRAKURIKULER
SMAN 1 GORONTALO

========================================================================
1. GAMBARAN UMUM APLIKASI
========================================================================

Aplikasi ini digunakan untuk mengelola kegiatan ekstrakurikuler sekolah
secara online. Aplikasi dapat dipakai oleh 4 jenis pengguna:

1. Admin
2. Siswa
3. Pembina
4. Kepala Sekolah

Fungsi utama aplikasi:

- Menampilkan daftar ekstrakurikuler ke publik
- Pendaftaran ekstrakurikuler oleh siswa
- Verifikasi pendaftaran oleh admin atau pembina
- Pengelolaan jadwal kegiatan
- Pencatatan kehadiran siswa
- Pencatatan penilaian atau prestasi siswa
- Pengumuman kegiatan
- Laporan peserta, jadwal, kehadiran, dan penilaian


========================================================================
2. ALAMAT HALAMAN YANG PENTING
========================================================================

Halaman umum:

- Beranda / landing page
- Informasi sistem
- Pengumuman
- Detail ekstrakurikuler
- Login
- Register siswa

Setelah login, pengguna akan otomatis diarahkan ke dashboard sesuai role:

- Admin -> Dashboard Admin
- Siswa -> Dashboard Siswa

3. PENJELASAN SETIAP ROLE
========================================================================

3.1. ADMIN

Admin adalah pengelola utama aplikasi.

Tugas admin:

- Mengelola akun pengguna
- Mengelola data siswa
- Mengelola data pembina
- Mengelola data ekstrakurikuler
- Menambah atau menghapus pengumuman
- Memeriksa pendaftaran siswa
- Menyetujui atau menolak pendaftaran
- Melihat laporan peserta
- Melihat laporan jadwal
- Melihat laporan kehadiran
- Melihat laporan penilaian
- Export data pendaftaran dan laporan

Menu penting admin:

- Dashboard
- Users
- Students
- Coaches
- Extracurriculars
- Announcements
- Registrations
- Participants
- Schedules
- Attendances
- Assessments
- Profile


3.2. SISWA

Siswa adalah pengguna yang mendaftar ekstrakurikuler dan melihat hasil
pembinaan.

Tugas siswa:

- Membuat akun sendiri
- Login ke sistem
- Melihat daftar ekstrakurikuler
- Membaca detail ekstrakurikuler
- Mendaftar ke ekstrakurikuler
- Melihat status pendaftaran
- Melihat jadwal kegiatan yang diikuti
- Melihat riwayat kehadiran
- Melihat penilaian atau prestasi
- Memperbarui data profil

Menu penting siswa:

- Dashboard
- Extracurriculars
- Registrations
- Schedules
- Attendances
- Assessments
- Profile


3.3. PEMBINA

Pembina adalah pengguna yang menangani ekstrakurikuler tertentu.

Tugas pembina:

- Melihat daftar ekstrakurikuler yang dibinanya
- Melihat peserta ekstrakurikuler
- Melihat pendaftaran siswa
- Menyetujui atau menolak pendaftaran siswa
- Menambah jadwal kegiatan
- Mengubah jadwal kegiatan
- Menghapus jadwal kegiatan
- Mengisi kehadiran siswa
- Menambah penilaian siswa

- Menghapus penilaian siswa
- Membuat pengumuman

Menu penting pembina:

- Dashboard
- Extracurriculars
- Participants
- Registrations
- Schedules
- Attendances
- Assessments
- Announcements
- Profile


3.4. KEPALA SEKOLAH

Kepala sekolah berfungsi sebagai pemantau atau pengawas.

- Melihat data kehadiran

- Profile


========================================================================
4. ALUR SISWA MENDAFTAR DARI AWAL
========================================================================

Ini adalah bagian paling penting untuk orang awam.

Langkah 1. Buka website

- Siswa membuka website aplikasi
- Siswa melihat daftar ekstrakurikuler yang tersedia di halaman depan

Langkah 2. Baca informasi ekstrakurikuler

- Siswa bisa klik salah satu ekstrakurikuler
- Siswa membaca:
  - nama ekstrakurikuler
  - deskripsi
  - syarat
  - jadwal latihan
  - informasi tambahan
  - pembina

Langkah 3. Klik tombol daftar

- Jika siswa belum login, sistem akan meminta login
- Jika siswa belum punya akun, siswa harus register dahulu

Langkah 4. Register akun siswa

Data yang diisi siswa saat register:

- Nama lengkap
- Email
- Password
- Konfirmasi password
- Jenis kelamin
- Tanggal lahir
- Nomor telepon
- Alamat
- Nama orang tua atau wali
- Nomor telepon orang tua atau wali

Catatan penting:

- Saat register, role akun otomatis menjadi siswa
- Sistem juga otomatis membuat data siswa
- NIS dan kelas bisa saja belum diisi pada saat awal

Langkah 5. Login otomatis atau login manual

- Setelah register berhasil, siswa biasanya langsung masuk ke sistem
- Jika siswa sudah punya akun, cukup login dengan email dan password

Langkah 6. Lanjutkan pendaftaran ekstrakurikuler

- Jika siswa tadi memulai dari halaman detail ekstrakurikuler, sistem akan
  mengarahkan kembali ke ekstrakurikuler tersebut
- Siswa klik daftar atau kirim pendaftaran
- Siswa boleh menambahkan catatan bila diperlukan

Langkah 7. Status menjadi menunggu verifikasi

Setelah pendaftaran dikirim:

- Status pendaftaran menjadi Pending
- Siswa harus menunggu verifikasi admin atau pembina

Langkah 8. Admin atau pembina memverifikasi

Kemungkinan hasil verifikasi:

- Approved -> siswa diterima
- Rejected -> siswa ditolak
- Pending -> masih menunggu

Langkah 9. Jika disetujui

Jika status Approved:

- Siswa dianggap menjadi peserta ekstrakurikuler
- Jadwal kegiatan akan terlihat di menu siswa
- Kehadiran dan penilaian siswa bisa dicatat oleh pembina

Langkah 10. Siswa memantau perkembangan

Siswa dapat melihat:

- Status pendaftaran
- Jadwal kegiatan
- Riwayat kehadiran
- Nilai atau prestasi
- Notifikasi pada dashboard

10. PENJELASAN STATUS PENTING
========================================================================

Status pendaftaran:

- pending  = masih menunggu verifikasi
- approved = diterima
- rejected = ditolak

Status kehadiran:

- present    = hadir
- absent     = tidak hadir
- sick       = sakit
- permission = izin

Jenis penilaian:

- assessment  = penilaian umum
- achievement = prestasi


========================================================================
11. CARA GANTI GAMBAR EKSTRAKURIKULER
========================================================================

Cara normal:

1. Login sebagai admin
2. Buka menu Extracurriculars
3. Klik edit pada ekstrakurikuler
4. Pilih gambar baru
5. Simpan

Catatan penting untuk hosting shared:

- Setelah perbaikan path hosting, upload manual dari admin harus masuk ke
  folder public_html/uploads/extracurriculars
- Jika gambar tidak muncul, biasanya masalah ada pada:
  - file upload belum masuk ke folder publik yang benar
  - image_path di database tidak sesuai
  - cache belum dibersihkan


========================================================================
12. MASALAH YANG SERING TERJADI
========================================================================

Masalah 1. Siswa tidak bisa login

Penyebab:

- email salah
- password salah
- akun tidak aktif

Solusi:

- cek email dan password
- cek status akun di admin


Masalah 2. Siswa sudah daftar tapi belum masuk peserta

Penyebab:

- status pendaftaran masih pending

Solusi:

- admin atau pembina harus menyetujui pendaftaran


Masalah 3. Gambar ekstrakurikuler tidak tampil

Penyebab:

- image_path salah
- file tidak ada di folder publik
- gambar diupload ke folder yang salah

Solusi:

- cek database kolom image_path
- cek file ada di public_html/uploads/extracurriculars atau
  public_html/images/extracurriculars
- jalankan clear cache


Masalah 4. Data lokal dan hosting berbeda

Penyebab:

- git push hanya memindahkan kode
- data database tidak ikut pindah otomatis

Solusi:

- export database lokal lalu import ke hosting
- atau input data langsung di hosting
